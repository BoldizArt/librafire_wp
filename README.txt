Dependencies

- Wordpress installed
- PHP 7.0 or greater version
- _s (underscores) theme installed

Setup/Run

- Clone this project from GitHub to PATH_TO_YOUR_WORDPRESS_WEBSITE/wp-content/themes folder
- Login as an administrator, go to the Appearance page and activate the LibraFire theme
- From the left admin menu, select the Employee content type, and add a new employee
- Fill all available fields and save te post
- Go to the preview
